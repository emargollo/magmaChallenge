## Reporting a bug

Please include the following in bug reports:

- steps required to reproduce the bug
- expected vs actual behavior
- versions of:
 - the challenge tool (ideally the commit SHA)
 - the JDK
 - the OS (Ubuntu 14.04, Windows 7...)
